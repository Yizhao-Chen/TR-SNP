computeGompertzCellTimings

Description

Data analysis function for computing individual cell timings from Gompertz parameters.

Usage

 computeGompertzCellTimings(gompar)

Arguments

gompar

a dataframe containing three sets of Gompertz parameters (a, b, k)

Details

This function computes cell timings using the parameters of Gompertz functions fitted on nELM, nLM, and nM.

Value

A data frame of at least 9 columns containing the cell number (N), and the computed values (tE, tL, tM, dE, dL and dT).
