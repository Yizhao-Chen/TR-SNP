overviewCellCountTable

Description

Check the format and provides a short summary of cell count data frame

Usage

   overviewCellCountTable(data)

Arguments

data

a data.frame containing cell count data.
This data.frame must contain 12 columns with the qualitative variables: Site, Year, Species, Tree, Sample, DY, RF, and the quantitive variables CZ, EZ, WZ, MZ, and PR. (see GRA).

Details

This function is designed to allow a first basic general cheking of cell count data, and print a short summary of factors and quantitative variables contained in the data frame.

Value

None
Information gathered by this function are directly printed in the console.
